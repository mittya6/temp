▼マティアス・ベルグ氏の実績
・パラリンピック/世界選手権で計27個のメダルを獲得
・弁護士資格
・プロのフレンチホルン奏者

これら複数のキャリア（マルチタスク）をどう両立させたのか？ ポイントは以下の3点だそうです。

① 人間は「同時に」複数のタスクをこなせない 
「マルチタスク」をしようとせず、一つ一つ丁寧にシングルタスクで処理するほうが、結果的に早くて確実。

② 毎晩10分間「明日の計画」を書き出す 
当日になるとやることを忘れがちなのでなので、前日にカレンダーを見ながら翌日の予定を正確に立てる。 
※ただし、突発的な仕事や連絡で予定は崩れる前提とし、「柔軟に対応できる余白タスク」を適度に挟んでおくのがコツ。

③ 休憩はこまめにとる 
長時間ぶっ続けで働き、帰宅後に「何もしたくない状態（脳の疲弊）」を作ると、ダラダラとネットを見るなど非効率な時間を過ごしがち。 
「1時間に5分休憩」など、仕事中こまめに回復を入れるほうがトータルで生産性が高い。
